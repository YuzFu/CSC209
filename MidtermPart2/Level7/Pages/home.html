The role of the <link> tag in the head of the iconBar.html file works for external css files. It connects to the css file and contribute to the style of the html document. In the iconBar.html page, for example, the iconBar.css file determines the color and size of the icon bar blocks, the color and size of the symbolds, the style when the icon is hovered or active.
